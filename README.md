#WarehouseMaster

Система управления складом на Java. Позволяет вести учёт товаров, управлять поступлениями и отгрузками, отслеживать остатки на складе.

## Описание

WarehouseMaster — приложение для автоматизации складского учёта. Основные возможности:

- Добавление, удаление и редактирование товаров
- Учёт поступлений и отгрузок
- Поиск и фильтрация товаров по различным критериям
- Контроль остатков на складе
